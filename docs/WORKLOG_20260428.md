## 关键问题根因（今日确认）

- “任务查询失败”主要是任务 ID 语义混用（worker task id vs video task id）。
- “查看按钮无动作”主要是任务有 item/output_data 但 `task_outputs` 为空时无回放输出。
- “问进度却重新生成”主要是模型工具调用被直接执行，缺少查询意图硬闸门。
- “预览图 XML 报错”是上游返回无效 bucket URL，不是前端组件崩溃。
