# 海盟会 - UI/UX 优化方案

## 一、设计风格定位

### 1. 设计理念

```
品牌调性：专业、智能、高效、现代
关键词：AI 驱动、视频创作、企业级、简洁
```

### 2. 色彩系统（红色系主题）

```css
/* 主色调 */
--primary: 红色系 (oklch 0.55-0.65, chroma 0.15-0.20, hue 15-25)
  - 主色：活力红 #E53935 / oklch(0.62 0.18 20)
  - 浅色：珊瑚红 #FF8A80
  - 深色：深红 #B71C1C

/* 辅助色 */
--accent: 橙红色渐变
--success: 绿色系
--warning: 琥珀色系
--error: 深红色系

/* 中性色 */
--background: 浅灰白
--foreground: 深灰黑
--muted: 中灰色
```

### 3. 字体系统

```
主字体：Noto Sans SC（已配置）
标题：700-900 字重
正文：400-500 字重
辅助：300-400 字重
```

## 三、交互体验优化

### 1. 加载状态

```tsx
// 骨架屏组件
<Skeleton className="h-24 w-full" />

// 按钮加载状态
<Button disabled={loading}>
  {loading ? <Loader2 className="animate-spin" /> : "生成视频"}
</Button>

// 页面加载
<PageLoader />
```

### 2. 空状态

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                      ┌─────────┐                            │
│                      │  图标   │                            │
│                      └─────────┘                            │
│                                                             │
│                    暂无数据                                  │
│                点击开始创建你的第一个视频                     │
│                                                             │
│                  ┌──────────────┐                           │
│                  │   开始创建   │                           │
│                  └──────────────┘                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3. 错误处理

```
Toast 通知：
- 成功：绿色，自动消失
- 警告：橙色，需手动关闭
- 错误：红色，显示详情

表单错误：
- 字段下方红色提示
- 提交按钮禁用
```

### 4. 操作确认

```
危险操作（删除、清空）：
- 二次确认弹窗
- 输入确认文字

重要操作（生成视频）：
- 资源消耗提示
- 倒计时确认
```

## 四、移动端适配

### 1. 响应式断点

```css
/* 移动端优先 */
sm: 640px   /* 小屏手机 */
md: 768px   /* 平板 */
lg: 1024px  /* 小屏电脑 */
xl: 1280px  /* 大屏电脑 */
2xl: 1536px /* 超大屏 */
```

### 2. 移动端特殊处理

```
- 底部导航栏替代侧边栏
- 触摸友好的按钮尺寸（最小 44px）
- 简化的表单布局
- 下拉刷新
- 无限滚动
```
