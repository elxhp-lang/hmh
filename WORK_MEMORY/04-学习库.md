# 学习库板块

## 核心功能
- 视频上传（用户上传优秀视频供学习）
- 智能视频分析（视觉模型分析内容）
- 多维度分析（风格、场景、镜头、色彩）
- 向量嵌入语义搜索
- 创作知识提取

## 技术栈
- **视觉模型**: `doubao-seed-1-6-vision`
- **向量嵌入**: Embedding Client
- **存储**: TOS 学习库目录 + Supabase

## 分析维度
1. 视频类型识别
2. 场景分析
3. 镜头语言
4. 色彩风格
5. 创作技巧
6. 商品展示分析
