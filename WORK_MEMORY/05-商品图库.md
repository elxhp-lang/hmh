# 商品图库板块

## 核心功能
- 商品图片上传
- 商品命名和描述
- 多视角图片管理
- 自动整合多视角图片
- 创意小海自动识别商品

## 存储结构
```
users/{user_id}/products/{product_id}/images/
```

## 工作流程
1. 用户上传商品图片并命名
2. 系统存储多视角图片
3. 自动整合为单一图片并标注
4. 创意小海对话时自动识别商品名称
5. 从商品库获取参考信息

## 关键决策记录

| - | 自动整合多视角图片 | 供 Seedance 2.0 参考 |
