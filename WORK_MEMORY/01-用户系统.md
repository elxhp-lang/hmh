# 用户系统板块

## 核心功能
- 用户注册（简化流程：仅需用户名和密码）
- 登录认证（JWT Token）
- 权限管理（super_admin/admin/finance/member）
- 注册审核流程（新用户需管理员审核）
- 飞书通知（注册成功后通知管理员）

## 权限体系
| 角色 | 权限等级 | 权限范围 |
|------|---------|---------|
| super_admin | 100 | 所有权限，包括用户管理 |
| admin | 80 | 用户管理、所有数据查看 |
| finance | 60 | 账单查看、发票管理 |
| member | 20 | 个人数据操作 |

## 关键决策记录
| 日期 | 决策 | 原因 |
|------|------|------|
| - | 第一个用户自动成为超级管理员 | 部署方无需手动创建 |
| - | 注册仅需用户名和密码 | 简化流程 |
| - | 新用户状态为 pending | 需管理员审核 |
