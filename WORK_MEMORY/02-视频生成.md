# 视频生成板块

## 核心功能
- 文生视频（Text-to-Video）
- 图生视频（Image-to-Video）
- 视频延长
- 音频生成
- 多模态参考（图片/视频/音频）

## 技术栈
- **模型**: Seedance 2.0
  - `doubao-seedance-2-0-260128` (标准版)
  - `doubao-seedance-2-0-fast-260128` (快速版)
- **存储**: TOS 对象存储（hmhv 桶）
- **凭证**: ARK_API_KEY（专门用于视频生成）

## 存储结构
```
hmhv/
├── users/{user_id}/
│   ├── videos/        # 用户生成的视频
│   ├── images/        # 用户上传的图片
│   └── temp/          # 临时文件
└── system/            # 系统资源
```

## 关键决策记录
| 日期 | 决策 | 原因 |
|------|------|------|
| - | 视频生成后自动存储到 TOS | 持久化，避免临时 URL 失效 |
| - | 按用户隔离存储 | 数据安全和管理 |
| - | 凭证分离（ARK_API_KEY vs VOLCENGINE） | 不同服务使用不同凭证 |

- `src/app/api/video/generate/route.ts` - 视频生成 API
