# 真人素材库 - 完整技术方案（公用人人素材库）

> 最后更新：2026-04-15

---

## 一、概述

### 1.1 功能定位
平台公用的真人演员素材库，所有用户共享。管理员录入方舟平台的真人演员素材，供所有用户在视频生成时引用。

### 1.2 核心能力
| 入口 | 功能 | 权限 |
|------|------|------|
| `/video` | 手动选择生成 | 所有用户 |
| `/agent/create` | 创意小海调用 | 所有用户 |
| `/admin/real-assets` | 素材管理 | 管理员 |

---

## 二、数据库设计

### 2.1 表结构（公用，无用户隔离）

```sql
CREATE TABLE real_assets (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  asset_id TEXT NOT NULL UNIQUE,              -- 方舟 Asset ID (如: asset-20260417010659-xkqtr)
  asset_url TEXT,                            -- 预览画面URL
  name TEXT NOT NULL,                        -- 素材名称 (如: "小明演员")
  description TEXT,                          -- 详细描述
  category TEXT,                             -- 分类 (如: "古装演员")
  status TEXT DEFAULT 'active',              -- 状态: active / inactive
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- 索引
CREATE INDEX idx_real_assets_name ON real_assets(name);
CREATE INDEX idx_real_assets_category ON real_assets(category);
CREATE INDEX idx_real_assets_status ON real_assets(status);
```

### 2.2 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | UUID | 是 | 主键 |
| asset_id | TEXT | 是 | 方舟 Asset ID，全局唯一 |
| asset_url | TEXT | 否 | 预览画面URL（用户在火山后台获取） |
| name | TEXT | 是 | 素材名称（如"小明演员"） |
| description | TEXT | 否 | 详细描述 |
| category | TEXT | 否 | 分类标签 |
| status | TEXT | 是 | active / inactive |
| created_at | TIMESTAMP | 是 | 创建时间 |
| updated_at | TIMESTAMP | 是 | 更新时间 |

## 三、API 设计

### 3.1 接口列表

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/api/real-assets` | 获取所有有效素材 | 所有用户 |

## 八、注意事项

1. **asset_id 唯一性**：全局唯一，不与用户绑定
2. **asset:// 前缀**：存储时存原始 ID，调用时拼接 `asset://` 前缀
3. **asset_url**：用户在火山后台获取的预览图 URL，用于管理页面展示
4. **素材状态**：支持禁用/启用，方便临时下线素材
