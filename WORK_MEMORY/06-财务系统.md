# 财务系统板块

## 核心功能
- 账单查询
- 发票申请
- 消费统计
- 财务数据导出

## 技术栈
- **火山引擎财务 API**: `@volcengine/openapi`
- **凭证**: VOLCENGINE_ACCESS_KEY_ID / SECRET_ACCESS_KEY

## 权限要求
- finance 角色可访问
- super_admin 可查看财务概览
