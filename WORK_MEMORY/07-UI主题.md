# UI主题板块

## 设计风格
- **风格**: Minimalism & Swiss Style（极简主义）
- **主色调**: 深蓝色系（#1E40AF / #3B82F6）
- **参考**: 火山引擎官网风格

## 核心规范
- 浅色背景必须使用深色字体
- 深色背景必须使用浅色字体
- 圆角 8-12px
- 所有用户可见文本必须是中文
- 禁止使用蓝紫色渐变（AI 味）

## 技术栈
- **UI 组件**: shadcn/ui
- **样式**: Tailwind CSS 4
- **主题**: Amber 主题
